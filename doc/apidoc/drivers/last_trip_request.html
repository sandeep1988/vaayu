GET /api/v1/drivers/:id/last_trip_request

Get latest available trip request. Will be deprecated when push notifications would be done

Supported Formats

json

Errors

Code Description
401 Unauthorized
403 Driver cannot access others data
404 Driver not found

Examples

{
      "id": 2,
      "status": "assign_requested",
      "trip_type": "check_in",
      "date": 1479106800
    }

Params

Param name Description
id
required

Validations:

  • Must be a number.
