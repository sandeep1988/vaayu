GET /api/v1/employee_trips/:id

Returns employee trip by id

Supported Formats

json

Errors

Code Description
401 Unauthorized
403 Employee cannot access others trip data
404 Employee trip not found

Examples

{
      "id": 1,
      "date": "2016-10-31T08:00:00.000Z",
      "trip_type": "check_in",
      "status": "created"
    }

Params

Param name Description
id
required

Validations:

  • Must be a number.
